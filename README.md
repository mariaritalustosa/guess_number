# Guess Number 

Guess Number é um projeto simples em Python onde o usuário define um número máximo e o sistema sorteia um número aleatório dentro desse intervalo. O objetivo é adivinhar qual foi o número escolhido pelo programa.

## Como funciona
1. O usuário informa o número máximo do jogo.
2. O programa sorteia um número aleatório.
3. O usuário tenta adivinhar o número.
4. A cada tentativa, o sistema informa se o número é maior ou menor.
5. Ao acertar, o jogo termina e mostra a quantidade de tentativas.
